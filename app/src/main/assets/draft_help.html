Draft Scout tries to evaluate the data (all matches so far) saved by scouting and produce a "ranking" of teams for Draft Alliance selection using the following calculations. You can vary/change the calculation by changing the modifiers for each item by using the 'Settings" feature in the Options Menu ( ⁝ ) in the Tool Bar at the top right.

Cubes

Scored
(A*((autoCubeSwitch + #Aextra + Xover(*.5))) + B*((autoCubeScale + #Textra + Xover(*.5))) + C*(teleCubeSwitch) + D*(teleCubeScale) + E*(teleOthr) + F*(teleExchange)) / # Matches
  • A - Multiplier
  • autoCubeSwitch - number of cubes scored in Switch during Autonomous
  • Aextra - Multiple cube(s) scored during Autonomous
  • Xover - Cross-Over to other side to Score
  • B - Multiplier
  • autoCubeScale - number of cubes scored in Scale during Autonomous
  • Textra - Multiple cube(s) scored during TeleOps
  • C - Multiplier
  • teleCubeSwitch - number of cubes scored in Switch during TeleOps
  • D - Multiplier
  • teleCubeScale - number of cubes scored in Scale during TeleOps
  • E - Multiplier
  • teleOthr - number of cubes scored in the the Other Alliance Switch
  • F - Multiplier
  • teleExchange- number of cubes put into the the Exchange
Collected
( G*(telePortal) + H*(telePwrZone) + I*(teleFloor) + J*(teleOthrFloor) + K*(teleRandomFloor) ) / # Matches
  • telePortal - number of cubes retrieved from Portal
  • telePwrZone - number of cubes retrieved from Power Cube Zone
  • teleFloor - number of cubes retrieved from our Platform Floor
  • teleOthrFloor - number of cubes retrieved from their Platform Floor
  • teleRandomFloor - number of random cubes retrieved from Floor

Cube Score is the sum of of Scored and Collected

Climbing

(L*(climbs) + (Lift1 *M) + (Lift2 *N) + (onPlatform *P) + (WasLifted *Q)) / # Matches
  • climbs - number of successful climbs by this robot
  • Lift1 - number of times this robot lifted one robot
  • Lift2 - number of times this robot lifted two robots
  • onPlatform - number of times this robot was on the platform in the last 30 seconds
  • WasLifted - number of times this robot was lifted by another team

Combined

( R*(climbScore) + S*(cubesScored) + T*(cubesCollected)) / # Matches
  • climbScore - total from the Climb score
  • cubesScored - total from the Cubes Scored score
  • cubesCollected - total from the Cubes Collected score

Switch

((A*((autoCubeSwitch + #Aextra + Xover(*.5))) + C*(teleCubeSwitch) + E*(teleOthr)) / # Matches
  • autoCubeSwitch
  • Aextra - Multiple cube(s) scored during Autonomous
  • Xover - Cross-Over to other side to Score
  • teleCubeSwitch
  • teleOthr

Scale

((A*((autoCubeScale + #Textra + Xover(*.5))) + C*(teleCubeScale) / # Matches
  • autoCubeScale
  • Textra - Multiple cube(s) scored during Autonomous
  • Xover - Cross-Over to other side to Score
  • teleCubeScale

Exchange

F*(teleExchange) / # Matches
